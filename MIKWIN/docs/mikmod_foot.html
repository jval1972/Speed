(1)

If MikMod library has been compiled with ALSA and/or EsounD support and without dynamic linking support, you'll have to add -lasound and/or -lesd as well.

(2)

Currently, MikMod's drivers don't have support for hardware mixing.

(3)

PAN_SURROUND will be mapped to PAN_MIDDLE if the library is initialized without surround sound, that is, if the variable md_mode doesn't have the bit DMODE_SURROUND set.
